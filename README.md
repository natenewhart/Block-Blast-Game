# Block Blast Game
This is a simple recreation of the Block Blast game, built using C++ and the SFML library.

## Getting Started
### Prerequisites
Make sure you have visual studio 2022 or later installed, along with the C++ packages.

### Installation / Building
The SFML library comes pre -installed in the \external folder
Clone project into visual studio and build to compile, the project file is already set up with all links and dependencies.

## Usage
Once the game has started use your mouse button to click and drag blocks onto the grid.
